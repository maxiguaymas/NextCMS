# NextCMS

A role-based content management system built with Next.js App Router.

## Features
- Role-based access (Admin, Editor, Viewer)
- Server-side authentication
- Public blog with SEO
- Clean architecture
- PostgreSQL + Prisma

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Prisma
- PostgreSQL (Neon)
- NextAuth

## Roles
- Admin: full access
- Editor: content management
- Viewer: read-only

```bash
npm install
npm run dev
